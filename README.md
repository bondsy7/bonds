# Daniel Bonds Portfolio

Cinematic portfolio starter built with Next.js App Router, TypeScript, Tailwind CSS, Framer Motion, and React Three Fiber.

## Run

```bash
npm install
npm run dev
```

## Structure

- Multi-page architecture: Home, About, Projects, Expertise, Playground, Hobbies, Contact
- Language toggle (DE/EN with flag UI)
- 3D hero scene (network/digital core)
- Project case study routes at `/projects/[slug]`
